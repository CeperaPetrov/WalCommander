				Wal commander 0.16.0


Описание:
	двухпанельный графический файл менеджер

Достоинтва:
	есть встроенный терминал и встроенный редактор и текстовый вьювер
	vfs: smb, ftp, sftp
	зависимость от дополнительных библиотек минимизирована 
		обязательны: libX11 
		опционально: libsmbclient 
		опционально: libssh2
		опционально: libfreetype //при ее отсутствии можно использовать только растровые шрифты x11
	
				
установка:
		убедиться в наличии библиотек
		запустить 
			make clean
				
			make 
			
			sudo make install
				
	ps:
		если при компиляции найдется нужная версия libsmbclient (dev) то
		скомпилируется с samba VFS
		аналогично с libssh2 (dev)
